# BB-8 Apps
Various tools built for Sphero BB-8 Droid.


## Android Information
- Min SDK: 19
- Target SDK: 26
- Java Version: 1.8
- Kotlin Version: 1.2.41


## Sections
### Notifications
Receive notifications on Android via Sphero BB-8 Droid.

### Voice Commands
Control Sphero BB-8 Droid on Android using voice commands.
